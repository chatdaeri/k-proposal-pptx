# 예시 02 — B2B DX 솔루션 도입 제안서

가상 "G회사" 대상 디지털 전환 솔루션 도입 9장 덱.

## 빌드

```bash
node ../../skills/proposal/scripts/build.cjs ./deck.cjs --no-lint
```

## 포함 슬라이드

1. 표지
2. 목차
3. 시장 성장 추이 (line chart + 표)
4. SWOT 2×2 매트릭스
5. 16주 4단계 로드맵 (content-grid)
6. AS-IS / TO-BE 비교
7. KPI 목표 (bar chart + 표)
8. SLA 운영 체계 (budget-table)
9. 마무리

## 사용 가이드

`CONTACT_DETAIL` 의 "G회사"를 실제 발주처 이름으로 교체하면 바로 사용 가능합니다.
