가상 "G러닝 GCON" 컨퍼런스 브랜드 경험 전략 8장 덱.

"Level Up" 메시지는 일반 명사이므로 그대로 사용했습니다.

## 빌드

```bash
node ../../skills/proposal/scripts/build.cjs ./deck.cjs --no-lint
```

2. 목차
3. 글로벌 컨퍼런스 트렌드 (bar chart + 표)
