# 예시 01 — 부동산 사업 검토 제안서

가상의 "S시 K동 123번지" 신축 오피스텔 분양성 검토 9장 덱.

모든 지명·회사명·수치는 가공 데이터입니다. 실제 프로젝트 정보로 교체해 사용하세요.

## 빌드

```bash
node ../../skills/proposal/scripts/build.cjs ./deck.cjs --no-lint
```

## 포함 슬라이드

1. 표지
2. 목차
3. 입지 현황 (시장 개요)
4. 경쟁 단지 비교 (bar chart + 표)
5. 평형 구성 제안 (content-grid)
6. 수익성 분석 (AS-IS/TO-BE)
7. 분양가 산정 (budget-table)
8. 리스크 검토 (content-grid)
9. 마무리
