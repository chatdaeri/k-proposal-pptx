# mckinsey-pptx — 한국어 데모 21종 ASCII 와이어프레임

> Claude Design 입력용. 슬라이드별 레이아웃 존·콘텐츠 슬롯·시각 계층만 추상화.
> 캔버스: 16:9 (96 char × 24 row 기준).
> 공통 크롬: 좌상단 Title + underline rule, 우상단 section_marker 박스, 하단 footer rule + copyright + page#.

범례:
```
═══   타이틀 underline (강조 라인)
───   하단 footer rule
▒▒    네이비/액센트 fill (헤더 박스, 강조 막대)
░░    light gray fill (보조 카드, secondary)
██    dark navy fill (full bleed)
···   placeholder (사용자 입력)
●     bullet
◆     diamond marker (key activity / interaction)
✓ ✗   pros / cons
[…]   슬롯 라벨
```
